Custom Confirmation Dialog

Test the beautiful confirmation dialog in different types

error_outline Danger Type

For destructive actions like delete

warning Warning Type

For actions that need caution

info Info Type

For informational confirmations

check_circle Success Type

For positive confirmations

Result:

Click any button above to test the dialogs

Code Example:

async function testDelete() {
  const confirmed = await showConfirmDialog({
    title: 'Delete Item?',
    message: 'This action cannot be undone.',
    confirmText: 'Delete',
    cancelText: 'Cancel',
    type: 'danger'
  });
  
  if (confirmed) {
    console.log('User confirmed!');
    // Proceed with deletion
  } else {
    console.log('User cancelled');
  }
}
